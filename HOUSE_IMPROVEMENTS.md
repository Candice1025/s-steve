# 房子改进 - 家具和装饰

## 新增功能

我已经大幅改进了 `buildAdvancedHouse` 方法，现在建造的房子包含：

### 🛋️ 家具

#### 客厅区域（前半部分）
- **工作台** - 放置在左侧，用于合成物品
- **熔炉** - 紧挨着工作台
- **箱子** - 放置在右侧，用于存储物品
- **书架墙** - 后左角有 2x2 的书架墙

#### 餐厅区域（中央）
- **餐桌** - 使用橡木栅栏作为桌腿，压力板作为桌面
- **椅子** - 4 把橡木楼梯围绕餐桌
- **地毯** - 餐桌下方铺设红色地毯

#### 卧室区域（后右角）
- **床** - 红色床铺
- **床头柜** - 使用木桶作为床头柜

### 💡 照明

- **墙壁火把** - 4 个火把分布在墙壁上
- **吊灯** - 天花板上悬挂灯笼（如果高度足够）
- **门廊灯** - 房子外面前门两侧的灯笼

### 🌸 装饰

#### 室内装饰
- **盆栽植物** - 前角落放置盆栽罂粟和蒲公英
- **地毯** - 餐厅区域铺设红色地毯

#### 室外装饰
- **花盆** - 窗户下方的花盆
- **门廊灯** - 前门两侧的灯笼照明
- **烟囱** - 如果使用石头材料，会在角落建造砖烟囱，内部有营火产生烟雾效果

### 🏠 外观改进

- **多个窗户** - 前后左右墙壁都有窗户，采光更好
- **精致的屋顶** - 层叠式屋顶设计
- **烟囱** - 石质房子会有砖烟囱
- **门廊照明** - 增强夜间美观度

## 方块统计

改进后的房子大约包含：
- **基础结构**: ~350 方块（墙壁、地板、屋顶）
- **家具和装饰**: ~50 方块
- **总计**: ~400 方块

## 使用的方块类型

### 家具方块
- `CRAFTING_TABLE` - 工作台
- `FURNACE` - 熔炉
- `CHEST` - 箱子
- `BOOKSHELF` - 书架
- `OAK_FENCE` - 桌腿
- `OAK_PRESSURE_PLATE` - 桌面
- `OAK_STAIRS` - 椅子
- `RED_BED` - 床
- `BARREL` - 床头柜
- `RED_CARPET` - 地毯

### 照明方块
- `WALL_TORCH` - 墙壁火把
- `LANTERN` - 灯笼
- `CAMPFIRE` - 营火（烟囱内）

### 装饰方块
- `POTTED_POPPY` - 盆栽罂粟
- `POTTED_DANDELION` - 盆栽蒲公英
- `FLOWER_POT` - 花盆
- `BRICKS` - 砖块（烟囱）

## 测试步骤

1. **重新编译**
   ```cmd
   .\gradlew.bat build
   ```

2. **启动游戏**
   ```cmd
   .\gradlew.bat runClient
   ```

3. **建造新房子**
   ```
   /steve spawn Builder
   /steve tell Builder build a house
   ```

4. **进入房子查看**
   - 走进房子内部
   - 查看所有家具和装饰
   - 测试功能性方块（工作台、熔炉、箱子、床）

## 自定义建议

如果你想进一步自定义，可以：

1. **更改家具位置** - 修改坐标偏移量
2. **添加更多家具** - 如酿造台、附魔台等
3. **更改装饰方块** - 使用不同颜色的地毯、床等
4. **添加二楼** - 增加楼梯和第二层
5. **扩大房子** - 增加 width 和 depth 参数

## 其他建筑类型

其他建筑类型（城堡、塔楼、谷仓等）保持原样。如果需要，我也可以为它们添加家具和装饰！

## 注意事项

- 家具会自动适应房子大小（基于 width、height、depth 参数）
- 小房子（5x5）可能空间有限，建议使用 9x9 或更大
- 所有家具都是功能性的，可以正常使用
- 照明充足，防止怪物在房子内生成
